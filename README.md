# Speech-to-Text Tool

A simple Python tool to convert audio files into text using Google's Web Speech API.

## Requirements

- Python 3
- Libraries listed in `requirements.txt`

## Installation

1. Create a virtual environment (optional but recommended):
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the script with the path to your audio file (WAV format recommended):

```bash
python transcribe.py path/to/audio.wav
```

The script will output the transcribed text.
